# 🌍 CodeAlpha - Language Translation Tool

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Artificial Intelligence Internship**.

The Language Translation Tool allows users to translate text between multiple languages using Google's translation service. It features a simple and user-friendly interface built with Python.

---

## 🚀 Features

- Translate text into multiple languages
- Automatic language detection
- User-friendly interface
- Copy translated text
- Translation history
- Download translated text

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Translate
- gTTS

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Stalin-coder/CodeAlpha_LanguageTranslationTool.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
